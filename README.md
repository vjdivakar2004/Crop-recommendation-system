# 🌾 Crop Recommendation System

A Machine Learning-based web application that helps farmers and agricultural enthusiasts identify the most suitable crop to cultivate based on soil and environmental conditions.

## 🚀 Live Demo

https://vj2004.pythonanywhere.com/

## 📖 Project Overview

The Crop Recommendation System predicts the best crop to grow using key agricultural parameters such as:

* Nitrogen (N)
* Phosphorus (P)
* Potassium (K)
* Temperature
* Humidity
* pH Value
* Rainfall

The system analyzes these inputs using a trained Machine Learning model and recommends the most suitable crop for cultivation.

## ✨ Features

* User-friendly web interface
* Machine Learning-based crop prediction
* Fast and accurate recommendations
* Supports multiple crop categories
* Deployed online using PythonAnywhere
